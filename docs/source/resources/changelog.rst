Changelog
=========

All notable changes to Mini-Arm.

[1.0.0] - 2025-XX-XX
--------------------

**Added**

- Complete documentation site with Sphinx
- Full API reference
- Hardware assembly guide with step-by-step instructions
- ROS2 Humble support with MoveIt2 integration
- Xbox controller teleoperation
- Face tracking example
- AI/LLM integration tutorials
- Bill of Materials with links
- 3D printing guide

**Changed**

- Migrated to Raspberry Pi Pico 2
- Updated CircuitPython to latest version
- Improved IK solver performance
- Enhanced trajectory interpolation

**Fixed**

- Gripper calibration issues
- Serial timeout handling
- Joint limit enforcement

[0.2.0] - 2024-XX-XX
--------------------

**Added**

- Cartesian control with onboard IK
- Trajectory execution with interpolation
- ROS2 package (beta)
- Basic visualization

**Changed**

- Refactored serial protocol
- Improved calibration routine

[0.1.0] - 2023-XX-XX
--------------------

**Initial Release**

- Basic joint control
- CircuitPython firmware for Pico
- Python client library
- 3D printable design files
- Simple examples
